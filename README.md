# ProductivityHacker
 
Productivity Hacker służy do śledzenia swoich nawyków (zarówno żywieniowych, w zakresie sportowym, w nauce...) oraz śledzenia zadań na dzień o czasu spędzonego na produktywności. Program został wykonany w formie gry - za wykonywanie zadań z listy Do Zrobienia, odznaczenie wykonania nawyku czy czas spędzony np. na nauce użytkownik dostaje punkty (xp), dzięki którym postać zdobywa wyższe poziomy i... nowe koszulki. Wyniki graficznie przedstawione są też przez wykres, porównujący stan xp oraz stan psychiczny/humor użytkownika
